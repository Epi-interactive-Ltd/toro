# Remove an animation route from a toro map.

Remove an animation route from a toro map.

## Usage

``` r
remove_route(map, route_id, settings = list())
```

## Arguments

- map:

  A toro map proxy object.

- route_id:

  A unique identifier for the route.

- settings:

  A list of settings for removing the route.

## Value

The updated map object.
