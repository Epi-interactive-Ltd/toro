# Pause a route animation on a toro map.

Pause a route animation on a toro map.

## Usage

``` r
pause_route(map, route_id, settings = list())
```

## Arguments

- map:

  A toro map proxy object.

- route_id:

  A unique identifier for the route.

- settings:

  A list of settings for pausing the animation.

## Value

The updated map object.
