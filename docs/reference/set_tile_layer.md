# Set the tile layer for the map.

Set the tile layer for the map.

## Usage

``` r
set_tile_layer(map, tiles)
```

## Arguments

- map:

  The map or map proxy object.

- tiles:

  A character vector of tile layer names. Options include "natgeo",
  "satellite", "topo", "terrain", "streets", "shaded", "lightgrey".

## Value

     The map or map proxy object for chaining.
